# 🌅 早霞晚霞预测系统

## 概述

专业的朝霞/晚霞拍摄条件预测工具，帮助摄影师捕捉最佳光线时刻。

## 预测算法

基于专业摄影师经验：

| 因素 | 评分逻辑 |
|------|----------|
| **高云（6000m+）** | ✅ 反射最佳，大幅加分 |
| **中云（2000-6000m）** | ✅ 散射效果好，加分 |
| **低云（0-2000m）** | ❌ 遮挡阳光，扣分 |
| **湿度（60-80%）** | ✅ 颜色更红艳，加分 |
| **日落后 15-20 分钟** | ✅ 最佳拍摄时间 |

## 输出格式

```

颜色：橙粉色
最佳时间：18:15-18:35
原因：低云较多，可能遮挡...
```
